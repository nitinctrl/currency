## Security Features

- Old passwords are automatically invalidated and stored in history
- Cannot reuse any of the last 5 passwords
- Account locks after 5 failed login attempts in 15 minutes
- Locked accounts require password reset via email
- Minimum password length: 8 characters
- All passwords are now strong with special characters

---

## Password Change Instructions

1. Login with the new password
2. Go to Settings > Change Password
3. Enter current password and new password
4. System will automatically logout and require re-login with new password
5. Old password becomes immediately invalid
